# projectRest

Angularjs project that consumes a Rest service.

## Setup Instructions.

We expect that you have installed on your machine:

```
Python
Node
Bower
Grunt
```

<br>

Navigate to projectRest directory.
<br>


For Windows users and if you rather not use python 2.x HTTPServer.
```
$ npm install
$ bower install
$ grunt serve
```
